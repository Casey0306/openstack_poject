В папке представлен файл setup.sh его можно использовать если будет необходимость установить
REST API приложение на новый сервер.

Инструкция по установке на Ubuntu 18.04
- Установка производится из пользователя root;
- Cкопироать файл setup.sh в корневую папку /root;
- Cделать права на исполнение файла chmod 755 setup.sh;
- Запустить файл setup.sh из корневой директории /root из под пользоателя root;
- Установка будет длится 4,5 минут;
- Обязательно после завершения установки сделать reboot на сервер!!! (Иначе не запустится web сервер)
